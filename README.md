## About

Course project built to practice Flutter fundamentals including state management, custom widgets, and navigation between screens.

## Features

- Gender selection (Male/Female)
- Height slider (120-220 cm)
- Weight and age controls with increment/decrement buttons
- BMI calculation with health interpretation
- Results page with personalized feedback
